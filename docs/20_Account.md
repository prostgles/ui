<h1 id="account"> Account </h1> 

Manage your account settings, security preferences, and API access.

<img src="./screenshots/account.svgif.svg" alt="Account Page" style="border: 1px solid; margin: 1em 0;" />

- **Account details**: View and update your account information.  
  - **Account information**: View all account details and associated data (workspaces, dashboards, views, etc.)  
- **Security**: Manage your account security settings.  
  - **Two-factor authentication**: Set up and manage two-factor authentication for enhanced security.  
    - **Generate QR Code**: Generate a QR code to set up 2FA with your authenticator app.  
    - **Can't scan QR code**: View manual setup instructions if you can't scan the QR code.  
    - **Confirm code**: Enter the code from your authenticator app to enable 2FA.  
    - **Enable 2FA**: Complete the 2FA setup process by confirming the code.  
    - **Base64 secret**: Manual setup secret key for your authenticator app.  
  - **Disable 2FA**: Turn off two-factor authentication for your account.  
  - **Change password**: Change your account password.  
  - **Active sessions**: View and manage your active web sessions.  
- **API**: View and manage your API access settings.  
  - **API Details**: View your API credentials and configuration.
