<h1 id="access_control"> Access control </h1> 

>  Not available on Desktop version
  
Manage user permissions and access rules for this database connection.
<img src="./screenshots/access_control.svgif.svg" alt="Access control" style="border: 1px solid; margin: 1em 0;" />

  - **Create Access Rule**: Add a new access control rule to define user permissions.  
  - **Save Access Rule**: Save changes to the current access control rule.  
  - **Cancel Changes**: Cancel editing the current access control rule.  
  - **Remove Rule**: Delete the selected access control rule.
